CSS kunskapssida

1.Vad är CSS? Hur används det?

Cascading style sheet. Används för att stilisera HTML kod. För att styra utseendet och layout av webbsidor.

2.Vad är skillnaden mellan inline-stil, inbäddad stil och extern stil i CSS?

Inbäddad CSS är i HTML och då riktar vi det mot element och beskriver vad vi vill göra med den. Inline är när man skriver CSS direkt i tagen. Extern innebär att CSS koden inte finns i HTML utan i dess egna CSS-dokument.

3.Vad är en selektor i CSS och hur används den?

Används för att rikta stilmallen till specifika element.

  1. enkla - väljer utifrån namn,id och klass
  2. kombinerade - väljer utifrån ett specifikt förhållande, till exempel " child" till ett parent element.
  3. pseudo klass - väljer utifrån vad som händer med ett element tex :hover
  4. pseudoelement - väljer del av element ex första bokstaven/linjen i p>
  5. attribut - väljer utifrån angivet attribut på element

4.Vad är skillnaden mellan klass-selektor och ID-selektor i CSS?

Id är ett specifikt namn, id bör bara existera en gång jämfört med klass som kan användas för flera element.

5.Vad är kaskadordningen i CSS och hur påverkar den hur stilar tillämpas på element?

En ordning för att avgöra vilka stilregler som ska tillämpas på ett element

  1. Arv - om element saknar regler så söks “föräldrars” stilregel och appliceras.
  2. relevans - behåller enbart regler som matchar element
  3. ursprung och betydelse- sorterar regler baserat på hur viktiga de är och vem som applicerat dem
  4. noggrannhet - väljer regler med högre specificitet om relevanta regler har samma ursprung och betydelse
  5. Ordning - om specifika regler har samma ursprung och betydelse väljer kaskadmodellen det senaste angivna värdet i källkoden.

6.Vad är responsiv webbdesign och hur kan CSS användas för att skapa responsiva layouter?

Man kan använda flexbox och grid layout. Det är att göra så att hemsidan ser bra ut och så att alla ska kunna navigera hemsidan.

7.Vad är boxmodellen i CSS och vilka element påverkas av den?

Alla HTML-element ses som boxar. Längst in i lådan har vi content. Contenten är wrappad av padding. Utanför padding så finns det border och efter border finns margin. Använder float och clear för att positionera. Margin frigör utrymme utanför lådan och padding gör det inuti lådan.

8.Vad är "grid layout" och hur skiljer det sig från "flexbox"?

Flexbox har inbyggt att man kan krympa/ växa element beroende på skärmstorlek. Flexbox kan bara gå i en riktig medans grid layout använder både rader och kolumner.

9.Vad är skillnaden mellan positionering med "static", "relative", "absolute" och "fixed" i CSS?

  1. Static - standard påverkas inte av regler
  2. Relative - positionerar relativt till sitt ursprungliga läge men påverkas av andra regler
  3. Absolute - positionerar relativt till det närmaste föräldrar elementet och den som fixed följer med scrollning.
  4. Fixed - fast position i förhållande till webbläsarfönstret och påverkas inte av rolling
