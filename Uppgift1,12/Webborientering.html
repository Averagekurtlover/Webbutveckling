Webborientering

1.Vad är en webbläsare och vad gör den?

Mjukvara används för att visa och interagera med webbsidor och webbapplikationer på internet.

2.Vad är en webbserver och vad gör den?

En dator eller mjukvara som lagrar, hanterar och levererar webbsidor och andra webbresurser till användare via internet. Skickar HTML, CSS och Javascript-kod till webbläsare.

3.Vad är en URL och vad innehåller den?

URL (Uniform Resource Locator) identifiera och hitta webbplatser som består av protokoll prefix ex som http eller https och domännamnet kan vara exempelvis youtube.com och ibland sökväg till specifik resurs (..../project/example2).

4.Vad är skillnaden mellan HTTP och HTTPS?

HTTP HyperText Transfer Protocol är det vanligaste protokollet för att överföra data mellan webbläsare och webbserver.HTTPS (Secure) är en säker version av HTTP där data krypteras för att skydda användarinformation och förhindrar obehörig åtkomst. HTTPS använder SSL/TLS (Secure Sockets Layer/Transport Layer Security)

5.Vilka är några exempel på frontend-ramverk och vad används de till?

För frontend-utveckling används vanligtvis tekniker såsom JavaScript, HTML och CSS tillsammans i ramverk som exempelvis React JS, Bootstrap och Vue JS. I backend-utvecklingen finns det ett flertal ramverk som hanterar databaser, processer, funktioner och liknande.
