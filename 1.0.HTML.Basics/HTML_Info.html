Vad är HTML och vad används det för?

HTML står för HyperText Markup Language och används för att skapa strukturen och innehållet på webbsidor. Det är ett märkspråk som använder taggar för att definiera olika delar av en sida.

Beskriv strukturen i ett HTML-dokument.

doctype, html, head, body, taggar

Vad är ett element? Ge exempel.

Definierar objekt i HTML. Webbläsaren utgår ifrån elementen för att tolka struktur

Vad är ett attribut? Ge exempel.

Anger ytterligare information om tagg

Vilka är de viktigaste skillnaderna mellan HTML5 och HTML4?

Annourlunda sätt att definiera innehåll. Html4 hade mer divs som fick id av attributen som nu finns i html5.

Vad är XML och XHTML?

XML är mer flexibelt än html och det används för dataöverföring. Xhtml mer strikt än html

Vad är DOM, och varför är den viktig?

Document objects modellen. Representation över hemsidans struktur. “Träd” formad hierarki där noder representerar tex element attribut. Ändra hemsidan.
