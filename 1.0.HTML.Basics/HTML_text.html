text level tags

      small är lvl -1 - ovikitgt, ger mer info men vill inte betona som copyright margianl anteckning
      bold är lvl 0 - mest visuellt
      italics lvl 1 - t.ex tekniska termer
      mark lvl 1,5 - text som ska higlightas
      empasis lvl 2 - betonar text
      strong lvl 3 - vissa att något är ytterst viktigt

citationer

cite titel på verk - tex böcker
inline citation - kortare citat, kan ha attributet cite för att länka till källa

blockquote används för längre citat

Förklarande text tags

      dfnanvänds när vi använder en definition som förklaras i kontexten
      abbr - när vi har en förkortning
